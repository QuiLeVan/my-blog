Mở đầu:

1 chương trình nhỏ trong swift, có tên & đuôi là .swift như: greeting.swift

```swift
func greet(name: String, surname: String) { print("Greetings \(name) \(surname)")
}
let myName = "Homer"
let mySurname = "Simpson"
greet(name: myName, surname: mySurname)
```
